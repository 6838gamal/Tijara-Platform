# **App Name**: Tijara Platform

## Core Features:

- Role Selection: Initial page allowing users to select their role (Admin, Merchant, Customer).
- Admin Dashboard: Admin panel with login, main dashboard, user management, store management, subscription management, payment management, notification management, coupon management, AI service management, general system settings, and technical support.
- Merchant Dashboard: Merchant panel with login/store creation, main dashboard, product management, category management, brand management, order management, customer management, coupon system, notification system, AI services, payment and shipping settings, reports and statistics, and store settings.
- Storefront: Public-facing store with a homepage, category pages, product detail pages, shopping cart, checkout page, order tracking, static pages, and customer login/account creation.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and reliability, reflecting the platform's core functions.
- Background color: Light gray (#F0F2F5), offering a neutral backdrop that emphasizes content and interface elements.
- Accent color: Indigo (#673AB7), to highlight key actions and interactive elements, ensuring clarity.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet approachable user experience.
- Consistent and professional icons throughout the platform.
- Clean, intuitive layouts with clear visual hierarchy across all dashboards and storefront pages.
- Subtle animations and transitions to enhance user engagement and provide feedback.
